# 🧠 Aptitude Master Web App

An interactive **multi-category aptitude quiz application** built using **HTML, CSS, and JavaScript**.
This app allows users to practice questions with customizable timers, hints, and dynamic question generation.

---

## 🚀 Features

🎯 Multiple Categories

* 📊 Quantitative
* 🗣️ English
* 🧠 Logical Reasoning
* 🔥 Mixed Mode

⏳ Custom Timer Selection (30 / 45 / 60 sec)

💡 Hint System

* Get hints for each question instantly

🎲 Dynamic Question Generation

* Auto-generates additional math questions

⚡ Real-Time Feedback

* Instant correct/wrong answer indication

🎨 Smooth UI

* Animated transitions and modern design

🔄 Restart Option after completion

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations & UI Design)
* JavaScript (Logic, Timer, DOM Manipulation)

## 🎮 How It Works

* Select a **category** and **timer**
* Answer questions within the time limit
* Use 💡 hints if needed
* Score is calculated at the end
* Restart anytime to practice again
